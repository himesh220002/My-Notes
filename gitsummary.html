Git & GitKraken Summary

Topic Description / Commands Notes
Branch Creation 
# Create a new branch
git checkout -b feature-branch
 Used to isolate new work from main branch.
Merging Branches 
# Merge branch into current branch
git merge feature-branch
 Combines code from different branches.
Push & Pull 
# Push changes
git push origin main

# Pull changes
git pull origin main
 Keeps local repo and remote repo in sync.
Useful Bash Commands 
# See all branches
git branch -a

# Check status
git status

# Log history
git log --oneline --graph
 Helps to inspect and track repo state.
